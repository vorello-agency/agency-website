# Tono de comunicación — Vorello Agency

> Guía operativa de voz y vocabulario para toda la comunicación de Vorello.
> Este documento complementa a `docs/brand-brief.md` y es la referencia obligatoria antes de redactar cualquier pieza de copy — web, redes, propuestas, emails o documentación.

---

## 1. Identidad de marca

| Campo                    | Valor                                                                                                     |
| ------------------------ | --------------------------------------------------------------------------------------------------------- |
| Nombre formal            | Vorello Agency                                                                                            |
| Nombre en comunicación   | Vorello                                                                                                   |
| Web                      | vorelloagency.com                                                                                         |
| Instagram                | @vorelloagency                                                                                            |
| Tagline principal        | Diseño. Tecnología. Producto.                                                                             |
| Frase de posicionamiento | Diseñamos y desarrollamos productos digitales bien pensados, visualmente cuidados y técnicamente sólidos. |

### Reglas de naming

- En comunicación pública se usa **"Vorello"** como nombre principal.
- **"Vorello Agency"** solo aparece como nombre formal: facturación, contratos, primer uso en un documento, bio de perfil.
- No usar "agency" ni "estudio" como descriptor genérico suelto. El posicionamiento lo lleva el tagline y el mensaje, no una etiqueta.
- Nunca escribir "Vorello agency" (minúscula) ni "VORELLO AGENCY" (todo mayúsculas).

## 2. Atributos del tono

Vorello debe sonar:

| Atributo        | Qué significa en la práctica                                  |
| --------------- | ------------------------------------------------------------- |
| **Claro**       | Frases cortas, estructura directa, sin rodeos                 |
| **Seguro**      | Afirmaciones concretas, sin condicionales innecesarios        |
| **Premium**     | Vocabulario cuidado, ritmo editorial, sin urgencia artificial |
| **Estratégico** | Conectar lo que hacemos con el resultado para el cliente      |
| **Directo**     | Ir al punto, sin introducciones largas ni relleno             |
| **Sobrio**      | Sin exageraciones, sin superlativos vacíos, sin hype          |

### Lo que NO es el tono de Vorello

| Evitar                        | Por qué                                              |
| ----------------------------- | ---------------------------------------------------- |
| Corporativo rígido            | Vorello es cercano, no distante                      |
| Informal / coloquial excesivo | No somos un freelancer casual                        |
| Vendedor agresivo             | Generamos confianza, no presión                      |
| Motivacional / inspirational  | No vendemos sueños, vendemos criterio y ejecución    |
| Técnicamente denso            | No hablamos para developers, hablamos para decisores |

| soluciones                           | Genérico, vacío. Todo el mundo "ofrece soluciones" |
| innovador / innovación               | Sobreusado, sin significado real                   |
| disruptivo                           | Cliché de startup. Vorello no "disrumpe"           |
| revolucionario                       | Exageración sin sustancia                          |
| siguiente nivel                      | Frase de agencia genérica                          |
| transformar / transformación digital | Buzzword corporativo                               |
| impulsar                             | "Impulsamos tu negocio" — vacío                    |

| empoderar                            | Fuera del tono de Vorello                          |

| holístico                            | Pretencioso y vago                                 |
| de vanguardia / cutting-edge         | Exageración innecesaria                            |
| best-in-class                        | Claim no verificable                               |
| world-class                          | Claim no verificable                               |
| líder / líderes en                   | Arrogante y vacío si no hay datos                  |
| 360°                                 | "Agencia 360" — exactamente lo que no somos        |
| full-service                         | Mismo problema que 360°                            |
| one-stop-shop                        | Vorello es selectivo, no generalista               |
| llave en mano / turnkey              | Suena a commodity                                  |
| experiencias únicas                  | Genérico y no verificable                          |
| apasionados                          | "Somos apasionados por..." — cliché                |
| rockstars / ninjas / gurús           | Fuera de tono, informal                            |

## 5. Frases prohibidas vs. alternativas

| ❌ Nunca decir                           | ✅ Decir en su lugar                                               |
| ---------------------------------------- | ------------------------------------------------------------------ |
| "Llevamos tu empresa al siguiente nivel" | "Diseñamos productos digitales con criterio y estándares altos"    |
| "Soluciones innovadoras y disruptivas"   | "Tecnología moderna bien aplicada"                                 |
| "Transformamos ideas en realidad"        | "Convertimos objetivos claros en sistemas digitales que funcionan" |
| "Tu agencia digital de confianza"        | "Vorello — Diseño. Tecnología. Producto."                          |
| "Impulsamos tu negocio digital"          | "Creamos productos digitales bien pensados"                        |
| "Somos apasionados por la tecnología"    | "Trabajamos con proceso, criterio y atención al detalle"           |
| "Creamos experiencias únicas"            | "Diseñamos interfaces cuidadas y funcionales"                      |
| "Hacemos páginas web"                    | "Diseñamos y desarrollamos productos digitales"                    |
| "Ofrecemos soluciones a medida"          | "Desarrollamos plataformas y sistemas a medida"                    |
| "Somos expertos en..."                   | "Trabajamos con..." / "Nos especializamos en..."                   |
| "Contáctenos para más información"       | "Hablemos de tu proyecto" / "Empecemos"                            |
| "No dude en contactarnos"                | "Escribinos" / "Hablemos"                                          |

| "Con más de X años de experiencia"       | (eliminar — no usar antigüedad como argumento)                     |

## 6. Patrones de estructura

### Estructura de frase preferida

**Acción + objeto + cualidad:**

> "Diseñamos productos digitales bien pensados, visualmente cuidados y técnicamente sólidos."

**Negación + afirmación:**

> "No construimos páginas aisladas. Construimos sistemas digitales pensados para funcionar y escalar."

**Lista rítmica (tres elementos):**

> "Diseño. Tecnología. Producto."
> "Criterio. Proceso. Calidad."
> "Bien pensado. Visualmente cuidado. Técnicamente sólido."

### Estructuras a evitar

| ❌ Patrón                         | Ejemplo                                           |
| --------------------------------- | ------------------------------------------------- |
| Superlativos sin sustancia        | "Los mejores sitios web del mercado"              |
| Preguntas retóricas de venta      | "¿Querés llevar tu negocio al siguiente nivel?"   |
| Promesas absolutas                | "Garantizamos resultados"                         |
| Urgencia artificial               | "¡No te quedes atrás!" / "Últimos lugares"        |
| Comparativos vagos                | "Mejor que la competencia"                        |
| Listas interminables de servicios | "Hacemos web, apps, SEO, ads, branding, redes..." |

## 8. Cómo hablar de Vorello en tercera persona

Cuando alguien más escribe sobre Vorello (prensa, partners, perfiles):

**Correcto:**

> "Vorello Agency diseña y desarrolla productos digitales bien pensados, visualmente cuidados y técnicamente sólidos."

> "Vorello se especializa en sitios web de alto rendimiento, ecommerce y plataformas digitales a medida."

**Incorrecto:**

> "Vorello Agency es una agencia digital líder que ofrece soluciones innovadoras para empresas."

> "Vorello Agency es un estudio creativo que transforma ideas en experiencias digitales únicas."

---

## 9. Cómo hablar de lo que NO hacemos

Vorello no ofrece como servicio:

- Gestión de redes sociales
- Campañas de ads (Meta, Google)
- Branding completo desde cero
- Diseño gráfico aislado
- Piezas sueltas sin contexto de producto

**Regla:** No mencionar estos servicios en ningún canal. Si un cliente pregunta, redirigir con claridad y sin disculparse:

> "Eso no es parte de nuestro enfoque, pero podemos recomendarte alguien que lo haga bien."

No decir:

> "Por el momento no ofrecemos ese servicio."
> "Estamos evaluando incluirlo."

## 10. Checklist de revisión de copy

Antes de publicar cualquier pieza de comunicación:

- [ ] No contiene palabras del vocabulario prohibido (sección 4)
- [ ] No contiene frases prohibidas (sección 5)
- [ ] Usa vocabulario aprobado (sección 3)
- [ ] El tono es claro, directo y premium — no vendedor ni genérico
- [ ] No promete servicios que Vorello no ofrece (sección 9)
- [ ] Usa "Vorello" en comunicación y "Vorello Agency" solo en contexto formal
- [ ] No tiene superlativos vacíos ni claims no verificables
- [ ] El CTA es claro, sobrio y no agresivo
- [ ] Se alinea con `docs/brand-brief.md`
- [ ] Pasa la "prueba de agencia genérica": ¿podrías reemplazar "Vorello" por cualquier otra agencia y el copy seguiría funcionando? Si sí, reescribir.

---

## 11. La prueba definitiva

> Si una frase podría aparecer en la web de cualquier agencia digital genérica, no es una frase para Vorello.

El copy de Vorello debe sonar específico, con criterio y con punto de vista. Si suena intercambiable, hay que reescribirlo.
